抖音视频口播文案提取工具

支持抖音长链、v.douyin.com 短链、完整分享口令文本；单条链接主流程仅使用第三方解析 API 返回的 mp4 视频地址。 查看诊断页

单个链接识别

可粘贴抖音链接、短链或完整分享口令；系统仅调用第三方解析 API，成功返回 mp4 地址后下载并转写。

识别结果会显示在这里。

上传视频/音频兜底

当链接下载失败时，可上传 mp4 / mp3 / wav / m4a 直接转写。

上传识别结果会显示在这里。

批量链接识别

每行一条，允许夹杂标题、话题、复制提示；单条失败不会影响其他链接。

上传 txt / csv / xlsx 读取链接

文件中可包含纯链接或完整分享文案，系统会自动提取抖音链接。

手动粘贴字幕/口播文本整理

粘贴抖音字幕、剪映识别字幕、第三方转写文本或人工粗稿；仅轻度纠错、去除明显重复语气词、补标点和断句，不总结、不改写、不改变原意。

该功能只依赖 OPENAI_API_KEY / OPENAI_BASE_URL / OPENAI_MODEL，不依赖 ASR。

解析接口配置说明

如果直接粘抖音链接失败，最稳定方案是配置第三方抖音解析 API。第三方接口负责把抖音链接解析成视频下载地址，本工具负责下载视频、提取音频、ASR 转文字、整理文案。不要把解析 API Key 写进代码，也不要提交 .env。

DOUYIN_RESOLVER_PROVIDER=api
DOUYIN_RESOLVER_API_URL=https://your-parser-api.com/parse
DOUYIN_RESOLVER_API_KEY=你的解析接口key
DOUYIN_RESOLVER_METHOD=POST
DOUYIN_RESOLVER_URL_FIELD=url
DOUYIN_RESOLVER_RESPONSE_VIDEO_FIELD=data.play_url
